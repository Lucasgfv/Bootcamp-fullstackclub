## I. Base do Projeto: TypeScript e Padronização

1.  **Inicialização:** `npm init -y`
2.  **Instalação de Dependências:** `npm install typescript @types/node prettier eslint -D`
3.  **Configuração Git:** Criar `.gitignore` (incluir `node_modules` e `dist`).
4.  **Configuração TS:** `npx tsc --init`
5.  **Ajuste no `tsconfig.json`:**
    * `rootDir`: "./src"
    * `outDir`: "./dist"
6.  **Estrutura Inicial:** Criar pasta `src/` e o arquivo `src/index.ts` com `console.log("Hello World");`.
7.  **Compilação:** Executar `npx tsc` para gerar a pasta `dist`.
8.  **Teste de Execução:** `node dist/index.js`

## II. Configuração do Servidor de Desenvolvimento

1.  **Gerenciador e Executor:** `npm install -D tsx` (Opcional: `npm install -g pnpm`).
2.  **Scripts no `package.json`:**
    ```json
    "scripts": {
      "dev": "tsx --watch src/index.ts",
      "build": "tsc"
    }
    ```

### Conceitos e Ajustes Importantes
* **Conventional Commits:** Ex: `git commit -m "chore: add typescript and tsx setup"`.
* **Documento `.npmrc`:** Criar na raiz com `save-exact=true`.
* **Linting:** Configurar `.eslintrc.json` e `.prettierrc`.
* **VS Code:** Criar `.vscode/settings.json` para ativar *format on save*.

## III. Backend: Fastify e Validação

1.  **Instalação Core:** `npm install fastify dotenv zod`
2.  **Integração Zod:** Utilizar `fastify-type-provider-zod`.
3.  **Documentação (Swagger):** `npm install @fastify/swagger @fastify/swagger-ui --save-exact`
4.  **Implementação:** Configurar `index.ts` registrando os plugins do Swagger e o Type Provider do Zod.
